# ⚡ AI Social Media Agent

> **Multi-Agent Content Generation System** — Automatically generate intelligent, platform-specific social media content for your brand.

---

## 🚀 Project Overview

**AI Social Media Agent** is a powerful multi-agent AI system that automatically generates social media content tailored to your brand. Simply provide your brand name, target audience, and campaign goal — the AI agents handle the rest.

The system uses multiple specialized AI agents that work together to:
- Build a comprehensive marketing strategy
- Conduct market research
- Write platform-specific posts (Instagram, LinkedIn, Twitter, Facebook, TikTok)
- Review and refine the generated content

---

## 🎯 Features

- 🤖 **Multi-Agent Architecture** — Dedicated agents for strategy, research, writing, and review
- 📱 **Multi-Platform Support** — Instagram, LinkedIn, Twitter, Facebook, TikTok
- 🎨 **Modern Cyberpunk UI** — Sleek neon-themed interface built with Streamlit
- 🎯 **Campaign Goals** — Brand Awareness, Lead Generation, Sales, Engagement
- ⚡ **One-Click Generation** — Full content pipeline triggered with a single button

## 🖥️ How to Use

1. Enter your **Brand Name** (e.g. "Cyber Security Academy")
2. Define your **Target Audience** (e.g. "Students and Professionals")
3. Select a **Campaign Goal** (Brand Awareness / Lead Generation / Sales / Engagement)
4. Choose your **Target Platforms** (Instagram, LinkedIn, Twitter, etc.)
5. Click **"⚡ GENERATE CONTENT"**
6. The AI agents will produce:
   - A tailored **Marketing Strategy**
   - In-depth **Market Research**
   - **Customized Posts** for each selected platform
   - An **AI Review** of every piece of content
